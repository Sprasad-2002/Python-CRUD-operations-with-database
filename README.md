# Python Database Connection (CRUD Operations)

## 📌 Project Description

This project demonstrates basic **database operations using Python**.
It shows how to connect Python with a database and perform **CRUD operations**.

CRUD stands for:

* **C** – Create
* **R** – Read
* **U** – Update
* **D** – Delete

This project is useful for beginners who want to understand how Python interacts with databases.

---

## 📂 Project Files

| File Name        | Description                         |
| ---------------- | ----------------------------------- |
| tablecreate.py   | Creates a new database table        |
| insertrecords.py | Inserts new records into the table  |
| showdata.py      | Displays all records from the table |
| updaterecord.py  | Updates existing records            |
| deleterow.py     | Deletes records from the table      |

---

## ⚙️ Technologies Used

* Python
* SQLite Database
* SQL Queries

---

## ▶️ How to Run the Project

1. Install Python on your system
2. Open the project folder
3. Run the Python files using terminal or VS Code

Example:

```
python tablecreate.py
python insertrecords.py
python showdata.py
python updaterecord.py
python deleterow.py
```
